# Ticket Master
Proyecto de Coder House para el curso de React JS. Ecommerce que esta enfocado en la venta de tickets de cine y teatro.

## Instrucciones
Para descargar y ejecutar el proyecto solo debe clonar el repositorio e instalar npm

```
npm install
```

Luego de tener el repositorio localmente se puede ejecutar con:

```
npm run start
```

## Ejecución
Al momento de correr la App se puede encontrar la página de inicio que posee un filtro de busqueda que trabaja sobre el listado de los nombres que coinciden con los busqueda realizada por el usuario. De igual manera, se posee una barra de navegación que redirecciona al usuario a las categorias de "peliculas" y "teatro", también al carrito de compras.

Cada categoria realiza el filtro sobre los elementos que coinciden con su tipo "peliculas" o "teatro", desplegando solo los que correspondan.

Cada item posee un botón que lleva a los detalles del show seleccionado, allí se pueden encontrar el tiempo de duración, genero, nombre y lenguaje. De igual manera es posible seleccionar el los detalles del show la cantidad de boletas, el boton de comprar solo será cliqueable si el número de boletas es mayor a 1.1
